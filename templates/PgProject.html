OUR MISSION

Ascii-art-web is a project that we are creating and running a server, in which it will be possible to use a web GUI (graphical user interface) version of our previous project, ASCII-ART.

Our webpage allows the use of the different banners:

  • shadow
  • standard
  • thinkertoy

Main Page Features:

The main page must include:

  • A box for text input.
  • Radio buttons to switch between banners.
  • A button to send a POST request to /ascii-art
  • An area to display the result.

HTTP Endpoints:

  • GET /: Sends HTML response, the main page.
  • POST /ascii-art: Sends data to Go server (text and a banner).

HTTP Status Codes

  • 200 OK: If everything went without errors.
  • 404 Not Found: If templates or banners are not found. (example: try a random url)
  • 400 Bad Request: For incorrect requests. (example: using non ASCII characters like æ¼¢ )
  • 500 Internal Server Error: For unhandled errors. (example: try selecting "stylize" option in the converter)
